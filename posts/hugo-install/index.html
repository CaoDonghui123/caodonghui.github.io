使用Hugo搭建网站步骤

1. 安装

我使用的系统是manjaro，在商店中就可以安装 也可以使用

$ sudo pacman -Syu hugo

2. 生成站点

$ hugo new site mysite

站点目录结构是：

.
├── archetypes #储存.md的模板文件，类似于hexo的scaffolds，该文件夹的优先级高于主题下的/archetypes文件夹
├── config.toml #配置文件
├── content #储存网站的所有内容
├── data #储存数据文件供模板调用
├── layouts #储存.html模板
├── static #储存图片,css,js等静态文件
└── themes #储存主题

  • 3. 创建文章

$ hugo new about.md         #生成about页面
$ hugo new post/first.md    #创建第一篇文章

  • 4. 安装皮肤

我选择的皮肤是 zozo

$ git clone https://github.com/varkai/hugo-theme-zozo themes/zozo

  • 5. 运行

$ hugo server

浏览器打开：http://localhost:1313
